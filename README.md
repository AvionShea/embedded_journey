## Learning Progression

This repo follows a deliberate, building-block approach:

- **Phase 1:** GPIO, timers, UART (getting comfortable with the hardware)
- **Phase 2:** Interrupts, PWM, ADC (real peripheral work)
- **Phase 3:** SPI, I2C, communication protocols (sensor integration)
- **Phase 4:** RTOS, real-time telemetry, IoT (production-level concepts)
